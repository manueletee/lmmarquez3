    Temática del Curso

    Tema 1. Modelado de Datos.



    Se estudia la evolución del lenguaje de modelado de datos (UML, Unified Modeling Language, por sus siglas en inglés), así como los elementos para la construcción de diagramas.

    Tema 2. Modelo Entidad-Relación

    Estudiar el concepto de modelo y el modelo entidad-relación (ER). Posteriormente analizar el ejemplo planteado, identificando las relaciones y entidades participantes, para su posterior diagramación.

    Tema 3. Instalando StarUML

    Descargar e instalar el programa StarUML, conocer las diferentes herramientas y utilidades.

    Tema 4. Diagramando en StarUML

    Descripción y Construcción del Modelo lógico mediante la matriz de relaciones (Entidades, Atributos y atributos clave) y el diagrama ER resultante (realizado en StarUML).
